# 常用命令

日常中常用的命令很多，有些隔段时间不使用可能就会遗忘了，现将其归纳整理，以便日后能快速回忆起。

# markdown的基础命令

#   一级标题
##  二级标题
### 三级标题
以此类推

>   一级引用的内容
>>  二级引用的内容
>>> 三级引用的内容

**加粗的内容**
*斜体的内容*
***粗斜体的内容***
~~删除的的内容~~
